# Para asistentes de IA

Si usas asistentes de IA como Claude, ChatGPT o Cursor, puedes darles el contexto completo de la documentacion usando estos archivos:

- [llms.txt](/llms.txt) - Indice de documentacion con links a todas las paginas
- [llms-full.txt](/llms-full.txt) - Documentacion completa en un solo archivo

## Uso

Copia la URL y pegala en el contexto de tu asistente de IA, o pidele que obtenga el contenido:

```
https://rut-kit.pages.dev/llms-full.txt
```

Esto ayuda a la IA a entender la API completa y dar ejemplos de codigo mas precisos.
