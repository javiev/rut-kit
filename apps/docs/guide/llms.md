# Para asistentes de IA

Si usas asistentes de IA como Claude, ChatGPT o Cursor, puedes darles el contexto completo de la documentación usando estos archivos:

- [llms.txt](/llms.txt) - Índice de documentación con links a todas las páginas
- [llms-full.txt](/llms-full.txt) - Documentación completa en un solo archivo

## Uso

Copia la URL y pégala en el contexto de tu asistente de IA, o pídele que obtenga el contenido:

```
https://rut-kit.pages.dev/llms-full.txt
```

Esto ayuda a la IA a entender la API completa y dar ejemplos de código más precisos.
